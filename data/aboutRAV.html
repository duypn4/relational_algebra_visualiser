1.Introduction:

Relational Algebra Visualizer is a java application which is used mainly to translate a relational algebra query into a query tree for IT learner. In addtion, the application is also able to hightlight inefficencies within the query as well as allows users to save the tree graph for later purposes. The tool has been developed by Ngoc Duy Pham for the final year project supervised by Dr. Floriana and Dr. David at University of Liverpool. In addition, a big thank to Dr Maduka who suggested me many helpfull ideas to accomplish this project.

2.Ethical Use of Data:

I have used a lexical analyser generator called JFlex by Klein, G., Rowe, S. and Decamps, R. I also made use of CUP which is LALR parser generator developed by TUM team and NativeSwing framework from Deckers, C. Moreover, there are some materials being studied to implement the project such as CUP documentation and code examples. JFlex/CUP article from University of Auckland in New Zealand.
